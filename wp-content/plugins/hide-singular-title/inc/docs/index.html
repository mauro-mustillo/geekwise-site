Hide Singular Title

A common question people ask when making a WordPress site is “How to remove/hide the “Page Title” on my homepage?”.

There are all sorts of good reasons why to want to do one of the above. A common one might be that you’re using a Page as your homepage instead of the default stream of posts that many themes use, and you don’t want a title getting in the way of your custom look. Another might be that you’re using a Page as a landing page, and again, the out-of-the-box title simply gets in the way.

So, do you want to hide titles for selective posts and pages without editing a single line of code? Then all you need to do is to obtain and install the Hide Singular Title plugin for WordPress.

Hide Title plugin lets you easily hide the title of any specified post or page. Don't want "Home" at the top of your homepage? Just switch the Hide Title option on.

Installation

Installation procedure is simple. Just follow instructions below.

  • Download the latest version of Hide Singular Title from myminiapp.com
  • Extract zip archive on your hard disk
  • Upload folder wp-hide-singular-title into your WordPress ./plugins folder with your favourite FTP app

  • Open WordPress Admin-> Plugins -> Installed Plugins
  • Click on Activate link on Hide Singular Title panel

If you haven’t installed free plugin WordPress Autoloader yet, you should get an admin notice:

Please click on the link WordPress Autoloader, then on Install Now button and follow installation instructions.

After installation please also activate the WP Autoloader plugin. (Visit WP Autoloader page)

After successful installation links Settings and CSS should appear. Plugin is ready for use now.

Change Global Settings

To change global settings of plugin please go to the WordPress Reading settings panel:

  • WordPress Admin -> Settings -> Reading

Alternatively you can use Setting link on plugins activation page.

On Reading Settings page you can alter global behaviour of plugin. Modify these settings according your needs. Please note that global settings will override your local settings.

Edit Post & Page Settings

You can change visibility of title of any singular post or page. Open editor for desired post or page and switch title on or off. Please note that global settings will override your local settings.

Press Publish/Update to save settings.

Top

Edit with Quick Edit

Alternatively you can use WordPress Quick Editor to switch title visibility on or off. Just open list of pages or post and click on Quick Edit on desired item.

Alter changes according your needs and press Update button.

Shortcodes

Hide Singular Title plugin comes with shortode to provide an option to enter title of post within content easily. Please note that shortcode ignores any filters applied to the titles. Additionally you can specify the tag and class names. This approach provides an easy way alternate the appearance of any page or post.

Regular code

[the_title]

Specify heading tag

[the_title tag="H1"]

Specify class and heading tag

[the_title class="mys-super-title" tag="H2"]
